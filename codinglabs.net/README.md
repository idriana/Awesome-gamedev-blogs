# Архив сайта codinglabs.net

Сodinglabs.net - Набор статей и туториалов по низкоуровневой разработке графических элементов игр, включая:
* Формат представления чисел с плавающей точкой IEEE 754;
* Гамма коррекция;
* Физически корректный рендеринг (PBR);
* Описание мирового пространства, пространства вида, пространства проекции, пространство модели и матриц перехода между ними (MVP matrices);
* Compute Shader'ы DirectX 11;
* Использование макросов C++ для динамической подгрузки кода вместо DLL;
* Управление пулом памяти;
* Использование теневой карты при отложенном рендеринге освещения;
* Пример отложенного рендеринга освещения на OpenGL;

Сайт создан разработчиком из Blizzard Entertainment Марко Аламиа. Сайт прекратил обновляться с 2014 года.

  * Медленным соединением

  * Наличием неработающих ссылок на сайте
